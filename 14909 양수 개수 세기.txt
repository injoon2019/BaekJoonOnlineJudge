문제
주어진 N개의 정수 중에서 양의 정수의 개수를 출력하는 프로그램을 작성하시오.

입력
첫째 줄에 최대 1,000,000개의 정수가 주어진다. 입력으로 주어지는 정수는 -1,000,000보다 크거나 같고, 1,000,000보다 작거나 같다.

출력
첫째 줄에 양의 정수의 개수를 출력한다.

예제 입력 1 
3 9 11 32 8 2 6
예제 출력 1 
7
예제 입력 2 
-2 0 21 3 8 17 32 -8 7 0
예제 출력 2 
6
예제 입력 3 
0
예제 출력 3 
0


#include <iostream>
#include <iomanip>
#include <string>
#include <algorithm>
#include <cmath>
#include <cstdlib>
#include <cstring>
#include <vector>
using namespace std;



int main(void)
{
	int number;
	int positive = 0;
	while (cin >> number;)
	{
		if (number > 0)
			positive++;
	}
	cout << positive << "\n";
	system("pause");
	return 0;
}
