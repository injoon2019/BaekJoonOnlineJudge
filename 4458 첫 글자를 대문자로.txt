문제
문장을 읽은 뒤, 줄의 첫 글자를 대문자로 바꾸는 프로그램을 작성하시오.

입력
첫째 줄에 줄의 수 N이 주어진다. 다음 N개의 줄에는 문장이 주어진다. 각 문장에 들어있는 글자의 수는 30을 넘지 않는다. 모든 줄의 첫번째 글자는 알파벳이다.

출력
각 줄의 첫글자를 대문자로 바꾼뒤 출력한다.



#include <iostream>
#include <iomanip>
#include <string>
using namespace std;


int main(void)
{
	int number;
	number = cin.get();
	cin.ignore();
	string sentence;
	for (int i = 0; i < number; i++)
	{
		getline(cin, sentence);
		if (sentence[0] >= 97 && sentence[0] <= 122)
		{
			sentence[0] = sentence[0] - 32;
		}
		cout << sentence<<"\n";
	}
	system("pause");
	return 0;
}
